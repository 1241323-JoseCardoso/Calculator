# Calculator Glossary

| **TEA (EN)**         | **TEA (PT)**        | **Description (EN)**                                                                                                                                                                          |
| :------------------- | :------------------ | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Calculator           | Calculadora         | An electronic device or software application used to perform arithmetic operations. It accepts numeric input through an entry pad, processes operations, and presents results on the display. |
| Display              | Ecrã                | The visual component of the calculator that shows the current input, intermediate values, results, or error messages.                                                                         |
| Entry Pad            | Teclado Numérico    | The collection of buttons used to enter digits, operators, and calculator commands.                                                                                                           |
| Button               | Botão               | An interactive control that performs a specific action when pressed, such as entering a digit, selecting an operator, or executing a command.                                                 |
| Digit                | Dígito              | A single numeric character (0–9) used to compose numbers entered into the calculator.                                                                                                         |
| Number               | Número              | A numeric value entered or produced by the calculator. A number may be an integer or a decimal value within the supported digit limit.                                                        |
| Operand              | Operando            | A numeric value on which an arithmetic operation is performed. An operation normally requires one or more operands.                                                                           |
| Operator             | Operador            | A mathematical symbol representing an arithmetic operation, such as addition (+), subtraction (−), multiplication (×), or division (÷).                                                       |
| Operation            | Operação            | A mathematical computation performed by applying an operator to one or more operands.                                                                                                         |
| Result               | Resultado           | The value produced after the successful execution of a calculation or operation.                                                                                                              |
| Calculation          | Cálculo             | The complete process of evaluating one or more arithmetic operations from user input to produce a result.                                                                                     |
| Sequential Execution | Execução Sequencial | The evaluation model in which operations are executed in the order they are entered, without applying mathematical operator precedence.                                                       |
| Current Input        | Entrada Atual       | The number currently being entered by the user and displayed before it is used in an operation.                                                                                               |
| Last Entry           | Última Entrada      | The most recent value entered by the user before selecting an operator or executing a calculation.                                                                                            |
| Pending Operation    | Operação Pendente   | An operation that has been selected but cannot yet be executed because the next operand has not been entered.                                                                                 |
| Previous Result      | Resultado Anterior  | The most recent calculation result stored by the calculator and available for subsequent operations.                                                                                          |
| Clear                | Limpar              | A command that clears the current input while preserving any pending operation or previously stored values.                                                                                   |
| Clear All            | Limpar Tudo         | A command that resets the calculator to its initial state by clearing all inputs, pending operations, stored values, and displayed results.                                                   |
| Error State          | Estado de Erro      | A condition indicating that the calculator cannot complete the requested operation. In this state, an error message is displayed until the user resets or clears the calculator.              |
| Digit Limit          | Limite de Dígitos   | The maximum number of digits that the calculator allows for displaying or entering a numeric value.                                                                                           |
| 8-Digit Maximum      | Máximo de 8 Dígitos | The calculator constraint that limits any displayed or entered number to a maximum of eight digits, excluding symbols such as the decimal separator or sign where applicable.                 |
| ERR                  | ERRO                | The error message displayed when the calculator detects an invalid operation, such as overflow or division by zero.                                                                           |
| Division by Zero     | Divisão por Zero    | An invalid arithmetic operation in which a number is divided by zero. The calculator enters the error state and displays an error message.                                                    |
| Overflow             | Overflow            | An error condition that occurs when the result of a calculation exceeds the calculator's supported numeric range or digit limit.                                                              |
| Decimal Number       | Número Decimal      | A numeric value containing a decimal separator, allowing representation of fractional quantities.                                                                                             |
| Full Expression      | Expressão Completa  | A mathematical expression containing multiple operands and operators entered by the user. In a sequential calculator, the expression is evaluated strictly in the order of entry.             |
